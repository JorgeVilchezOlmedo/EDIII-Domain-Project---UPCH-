# EDIII-Domain-Project---UPCH-
The surface of dengue virus is covered with 180 protein eelopes homodimers lying flat on the virion surface. Each monomer consist of three domains, of which ED3 contains critical neutralization determinants. The DENV3 Genotype changes the stability of ED3.
